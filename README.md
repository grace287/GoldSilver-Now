# 🟡 금은나우 (GoldSilver Now)

> **슬로건:** "지금, 금과 은의 온도."

한국금거래소 시세를 기반으로 **금·은 실시간 시세 조회**와 **변동 추이 시각화**를 제공하는 플랫폼입니다.  
크롤링 연습을 넘어 **금 데이터 SaaS / 데이터 기반 자산 플랫폼**을 지향합니다.

## 🎯 제품 개요

- **목적**: 한국금거래소 시세 데이터 기반 **금·은 실시간 시세 조회 + 변동 추이 시각화 플랫폼** 구축
- **가치**: 빠르고 직관적이며, 기록이 남는 시세 플랫폼

---

## 📌 문제 정의

| 사용자 문제 | 우리의 해결 |
|------------|------------|
| 금 시세를 자주 확인해야 한다 | 실시간에 가까운 주기적 업데이트 (5~10분) |
| 모바일에서 보기 불편하다 | 반응형·다크모드 지원 대시보드 |
| 과거 추이를 한눈에 보기 어렵다 | 7일/30일 차트 시각화 |
| 가격 도달 알림이 없다 | Phase 2에서 알림·구독 검토 |

---

## 👥 타겟 사용자

- **1차**: 금 투자자, 실물 금 보유자, 금은방 관계자
- **2차**: 환율/경제 관심 사용자, 단기 투자자

## 🔹 주요 API

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/prices/today` | 오늘 금/은 시세 반환 |
| GET | `/api/prices/history?metal=gold` | 기간별 시세 (차트용) |
| GET | `/api/prices/change-rate` | 전일 대비 변동률 |
