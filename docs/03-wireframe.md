## 2. 메인 페이지 (모바일)

```
┌──────────────────────┐
│ 🟡 금은나우    [☀/🌙] │
├──────────────────────┤
│  금 1돈               │
│  살 때  ₩385,000 ▲1.2%│
│  팔 때  ₩378,000 ▲0.8%│
├──────────────────────┤
│  [7일] [30일]         │
│  ┌────────────────┐  │
│  │   📈 차트       │  │
│  │   (세로 스크롤) │  │
│  └────────────────┘  │
├──────────────────────┤
│  은 시세             │
│  ₩4,250/g  ▲0.5%    │
├──────────────────────┤
│  [7일] [30일]        │
│  ┌────────────────┐  │
│  │   📈 차트       │  │
│  └────────────────┘  │
└──────────────────────┘
```

- 세로 스크롤 한 페이지에 금 블록 → 금 차트 → 은 블록 → 은 차트 순서.
- 숫자 폰트 크기 크게 유지 (가독성).

---

## 3. 컴포넌트 단위

| 영역 | 구성 요소 | 비고 |
|------|-----------|------|
| 헤더 | 로고, 슬로건, 다크모드 토글 | 고정 또는 스크롤 시 축소 |
| 금 시세 카드 | 제목, 살 때/팔 때 금액, 전일 대비 %, 기준 시각 | 카드 또는 섹션 |
| 금 차트 | 기간 탭(7일/30일), 라인 차트 | 차트 라이브러리 연동 |
| 은 시세 카드 | 제목, 단가, 전일 대비 %, 기준 시각 | 금과 동일 패턴 |
| 은 차트 | 기간 탭, 라인 차트 | 금과 동일 |
| 푸터 | 출처, 약관 | MVP 최소 |

---

## 4. 인터랙션

- **기간 탭**: 7일/30일 클릭 시 해당 기간 데이터로 차트 갱신 (API 재요청 또는 클라이언트 필터).
- **다크모드**: 토글 시 테마 전환, 로컬 저장(선택).
- **새로고침**: 페이지 로드 시 `/api/prices/today`, `/api/prices/history` 호출로 최신 표시.
- MVP에서 **드릴다운(상세 페이지)** 없음 — 메인 한 페이지로 충족.

---

## 5. 시각 가이드 (요약)

- **포인트 컬러**: 골드 계열 (#D4AF37 또는 프로젝트 토큰).
- **등락**: 상승 초록, 하락 빨강 (또는 프로젝트 규칙).
- **타이포**: 숫자 강조(큰 글씨), 본문은 읽기 쉬운 sans-serif.
- **차트**: 단색 라인, 그리드 얇게, 범례 최소.
